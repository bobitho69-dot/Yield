Open source builds

The desktop apps are built from the open desktop/ Electron project in the Yield repo, and published to GitHub Releases. Want to build your own signed copy?

cd desktop
npm install
npm run dist:mac    # → .dmg     (build on macOS)
npm run dist:win    # → .exe     (build on Windows)

See desktop/README.md for signing and notarization notes. Downloads above point to the latest published release.
