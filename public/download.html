Need a signed .dmg or .exe?

The installed app above is the full Yield experience and updates itself automatically — no downloads to manage. If you specifically need a signed native build, Yield ships an open-source Electron project you can build yourself:

desktop — zsh 
$ cd desktop
$ npm install
$ npm run dist:mac    # → .dmg   (build on macOS)
$ npm run dist:win    # → .exe   (build on Windows)

See desktop/README.md for signing and notarization. Native installers must be built on a Mac (for .dmg) and Windows (for .exe) — that's why they aren't hosted here yet.
