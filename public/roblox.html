Link Roblox Studio

Link your Studio to Yield once. After that, every place you open in Studio auto-syncs by its PlaceId — this list of games fills in automatically as you open them, with no naming or per-game pairing ever again.

1

Download the Yield plugin and drop the .lua file into your local Roblox Studio Plugins folder (Windows: %LOCALAPPDATA%\Roblox\Plugins, Mac: ~/Documents/Roblox/Plugins), then restart Studio.

2

Click Get link code below, open the Yield panel in Studio's toolbar, and paste the code in — just this once.

3

Allow the HTTP-request and script-injection prompts Studio shows the first time. From then on, open any saved/published place and it syncs automatically.
