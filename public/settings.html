Backend & secrets

Yield never stores your secrets. Your app's backend & API keys live in your own free Cloudflare Worker, connected to your GitHub — Yield writes the code, you add the secret values in Cloudflare.

  1. Store your code — connect GitHub (above).
  2. Make a Cloudflare account — free at cloudflare.com → Workers & Pages.
  3. Connect your repo — Create → Workers → Connect to Git → pick the app's repo. (The first connect can error — retry / re-authorize GitHub; it usually works the second time.)
  4. Ask Yield to build the backend — open the app in the builder and paste the prompt below.
  5. Add your secrets in Cloudflare — your Worker → Settings → Variables and Secrets → add each secret Yield listed → Deploy.
