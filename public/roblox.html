Link Roblox Studio

Link Studio to Yield once. After that, every place you open auto-syncs by its PlaceId — the chatbot can then read the whole game and build/edit it live.

1

Download the Yield plugin and drop the .lua file into your local Studio Plugins folder (Windows: %LOCALAPPDATA%\Roblox\Plugins, Mac: ~/Documents/Roblox/Plugins), then restart Studio.

2

Click Get link code, open the Yield panel in Studio's toolbar, and paste the code — just this once.

3

Allow the HTTP-request and script-injection prompts Studio shows the first time. From then on, open any saved place and it syncs automatically.

Sign in with Roblox

Marketplace Key (optional)

A Roblox Open Cloud API key (Assets Read+Write permission), added once for your whole account. Only used to publish AI-generated 3D models to your Roblox account when a mesh can't be imported straight into Studio — a rare fallback. Marketplace search doesn't need it: the Studio plugin searches, virus-scans, and inserts free models itself. Stored encrypted — never shown again, never sent back to your browser.

Create one at create.roblox.com → Creator Dashboard → Open Cloud → API Keys: add the Assets API system with Read and Write permissions for your creator (there is no "Toolbox" scope — Assets is the one that matters, and it's what enables 3D-model upload). The creator id is the numeric id of your account or group.
