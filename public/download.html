Install in one tap

Yield is a Progressive Web App, so it installs like a native app — no app store, no file to run. Pick your platform:

Chrome / Edge

Desktop
  1. Click Install Yield above — or the install icon (⊕ / a monitor with a down-arrow) at the right of the address bar.
  2. Confirm Install. Yield opens in its own window and lands in your Start menu / Applications / dock.

Safari (Mac)

macOS
  1. Open the app in Safari.
  2. Menu bar → File → Add to Dock… → Add. Yield now launches from your dock.

iPhone / iPad

iOS
  1. Open the app in Safari.
  2. Tap Share → Add to Home Screen → Add.

Android

Android
  1. Open the app in Chrome.
  2. Tap ⋮ → Install app (or Add to Home screen).

Want a native installer (.dmg / .exe)?

The installed app above is the full Yield experience and updates itself automatically — no downloads to manage. If you specifically need a signed native build, Yield ships an open-source Electron project you can build yourself:

cd desktop
npm install
npm run dist:mac    # → .dmg   (build on macOS)
npm run dist:win    # → .exe   (build on Windows)

See desktop/README.md for signing and notarization. Native installers must be built on a Mac (for .dmg) and Windows (for .exe) — that's why they aren't hosted here yet.
